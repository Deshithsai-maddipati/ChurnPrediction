About : 

This Project helps to predict the churn for the given Telecom Churn DataSet.

Major Steps :

Perform EDA on the data Set and prepared the clean Data Set.

Train Logistic-Regression Model,Random Forest Classifier on the Data prepared.

Evaluate the Model using precision,Recall,F1-Score and ROC Curve.

results : 
 
Achieved  a Precision of 0.62 and Recall of 0.57 with Logisitic Regression Model Which will be the good for the Requirement where we focus of people who will have higher chances of churning But Costs can be Increased as Unneccasary wastes on Loyal-Customers.

Achieved  a Precision of 0.66 and Recall of 0.43 with Random Forest Classifier Model Which will be the good when the Bussiness Budget is Tight.
